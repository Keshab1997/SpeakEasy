## 🛠️ Screenshot Taking Tips

1. **Use Android Studio / iOS Simulator** — এমুলেটরে অ্যাপ রান করে screenshot তুলুন
2. **Device Frame যোগ করুন** — [Screely](https://www.screely.com/), [Mockuphone](https://mockuphone.com/), বা [Cleanmock](https://cleanmock.com/) ব্যবহার করে device frame যোগ করুন
3. **কন্ট্রাস্ট চেক করুন** — টেক্সট যেন ব্যাকগ্রাউন্ড থেকে সহজে পড়া যায়
4. **Bangla Text ব্যবহার করুন** — স্ক্রিনশটে বাংলা টেক্সট থাকলে বেশি আকর্ষণীয় হবে টার্গেট অডিয়েন্সের জন্য
5. **Highlight Key Elements** — লাল/নীল বৃত্ত বা arrow দিয়ে important features highlight করুন
6. **Consistent Color Theme** — AppColors (Primary: #2563EB, Secondary: #10B981) অনুযায়ী consistent রাখুন
7. **Resolution** — 1080x1920 px বা 1242x2688 px (iPhone) রেজোলিউশনে তুলুন
