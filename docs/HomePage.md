# Home Screen Design Specification

## Create Home Screen for Spoken English Learning App

Use Flutter with Material 3 design.

## AppBar

Title:

```text
SpeakEasy
```

Right side icons:

* Notification icon
* Profile avatar

---

## Greeting Section

Show:

```text
Good Morning, User 👋
```

Below that:

```text
Keep practicing English every day.
```

---

## Progress Card

Display:

* Daily Streak
* Lessons Completed
* Progress Percentage

Example:

```text
🔥 7 Days Streak
📚 15 Lessons Completed
🎯 60% Progress
```

Use a beautiful card with rounded corners.

---

## Today's Word Card

Display:

Word:

```text
Beautiful
```

Meaning:

```text
সুন্দর
```

Example:

```text
She is a beautiful girl.
```

Add:

* Speaker button
* Favorite button

---

## Continue Learning Section

Horizontal cards.

Cards:

* Beginner English
* Grammar Basics
* Daily Conversation

Each card contains:

* Image
* Title
* Progress bar

Clicking a card should open Lesson Screen.

---

## Quick Practice Section

Grid with six items:

1. Vocabulary
2. Grammar
3. Conversation
4. Listening
5. Speaking
6. Quiz

Each item has:

* Icon
* Title

Use GridView with two columns.

---

## Daily Challenge Card

Show:

```text
Today's Challenge
Speak 10 English sentences
```

Add Start button.

---

## AI Teacher Banner

Display:

```text
Practice with AI Teacher
```

Button:

```text
Start Chat
```

Open AI Chat Screen.

---

## Achievement Section

Horizontal list.

Examples:

* Beginner Badge
* 7 Day Streak
* Vocabulary Master

---

## Bottom Navigation Bar

Tabs:

1. Home
2. Learn
3. Practice
4. AI Teacher
5. Profile

Current page:

Home

---

## UI Requirements

* Material 3
* Responsive layout
* Smooth scrolling
* Use SingleChildScrollView
* Use reusable widgets
* Modern card design
* Rounded corners
* Proper spacing
* Light and Dark theme support

Build production-quality Flutter code.
