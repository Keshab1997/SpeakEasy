# GitHub

GitHub is a DotCraft MCP plugin for working with repositories, issues, pull requests, code search, and related GitHub resources through GitHub's hosted MCP endpoint.

## Setup

Set the `GITHUB_PERSONAL_ACCESS_TOKEN` environment variable to a GitHub personal access token with access to the repositories you want DotCraft to use. Then install and enable the GitHub plugin in DotCraft.

This first version contributes only the GitHub MCP server configuration.
